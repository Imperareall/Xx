## 四、背景纹理与全局视觉体系

### 4.1 全局背景色

`index.css` 中设置：

```css
body {
  background: #0c0c0e;
  color: #a1a1aa;
}
```

### 4.2 环境光晕（Ambient Glow）

通过 `body::before` 和 `body::after` 固定伪元素实现，固定飘浮在页面上方/下方：

```css
body::before {
  position: fixed; top: -20%; left: -10%;
  width: 700px; height: 700px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(99,102,241,0.06) 0%, transparent 70%);
  filter: blur(140px);
  pointer-events: none;
  z-index: -1;
}
body::after {
  position: fixed; bottom: -20%; right: -10%;
  width: 800px; height: 800px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(139,92,246,0.05) 0%, transparent 70%);
  filter: blur(160px);
  pointer-events: none;
  z-index: -1;
}
```

**作用**：页面底层永远漂浮一层微弱的 indigo-purple 环境光，极大丰富暗色背景的纵深感。

### 4.3 胶片噪点（Film Grain）

在 `App.tsx` 最外层 div 内添加了 `<div className="film-grain" />`，CSS 定义：

```css
.film-grain {
  position: fixed; inset: 0; z-index: 9998;
  pointer-events: none;
  opacity: 0.025;
  mix-blend-mode: overlay;
  background-image: url("data:image/svg+xml,...feTurbulence...");
  background-repeat: repeat;
  background-size: 256px 256px;
}
```

**技术细节**：
- 使用 SVG `feTurbulence` 滤镜生成噪声
- `baseFrequency="0.82"` + `numOctaves="5"`
- `opacity: 0.025` 几乎不可见，仅为银盐胶片质感
- `mix-blend-mode: overlay` 使噪点与暗色背景融合

### 4.4 多米诺骨牌画廊背景光斑

在 GridView.tsx 的 3D 画廊区域还有两个缓慢漂浮的径向渐变光斑（indigo + purple），动画循环 18s/24s，进一步分层。

## 五、各页面功能详解

### 5.1 Memory 页面（`GridView.tsx`）

**顶部 Hero**：
- 全屏封面图（`微信图片_202606110934332.jpg`），`objectFit: contain`
- 毛玻璃居中卡片（subtitle + 标题 + Since）
- 底部 Scroll 指示器呼吸动画
- 背景视差上移效果（useScroll + useTransform）

**3D 多米诺骨牌画廊**：
- 引言："一整个宇宙 换一颗红豆"（Playfair italic, 15px）
- 9 张骨牌卡片从左到右以 `position: absolute` 排列
- 每张卡片有独特 3D 倾斜预设（rotation 数组，9 种不同 rotateX 值）
- 透视容器 `rotateX: 22deg rotateY: -15deg rotateZ: -5deg`
- **滚轮驱动**：画廊可横向滑动，原生 `el.addEventListener('wheel', ...)` 阻止页面上下滚
- **Hover 动效**：卡片上浮 60px + scale 1.1 + zIndex 9999，其余卡片 brightness 0.3 模糊
- 卡片半透明边框 `border: 1px solid rgba(255,255,255,0.05)`
- 阴影 `0_30px_60px_-15px_rgba(0,0,0,0.9)`，hover 时加深
- **点击**：触发 `openDetail(item.id)`，弹出 DetailView

**注意**：
- 不再使用 GridCard 组件（DominoCard 已内联在 GridView.tsx 中）
- GridCard.tsx 文件仍存在但可能未被引用

### 5.2 About 页面（`AboutSection.tsx`）

**纯图片滚动流，无文字内容**：

顺序排列 4 条横向自动滚动的图片条：

1. **Forever** — 向左滚动（73张，45s）
2. **experiencepic** — 向左滚动（21张，40s）
3. **xx** — 向右滚动（26张，36s）
4. **Pets** — 向右滚动（13张，30s）

每条图片流：
- 图片双份拼接实现无缝循环
- `AnimatePresence` + `x: ['0%', '-50%']`
- 左右渐变遮罩
- Hover 时 `scale: 1.05`
- 点击使用 ImageLightbox 多图浏览
- 背景色 `#0F1115`

### 5.3 Links 页面（`LinksSection.tsx`）

- 背景图：`/合照pic/微信图片_202606110934331.jpg`，`opacity: 0.28`
- 三个社交按钮：抖音 / 哔哩哔哩 / 小红书
- **3D 透视倾斜**：`perspective: 1000px` + `transformStyle: preserve-3d`
- 默认：`rotateY(-15deg) rotateX(6deg) skewX(-3deg)`，图标淡化银白
- Hover 弹簧回正 + 前推 `scale(1.12) z(40)`
- 各自霓虹色：抖音 `#00f2ea`、哔哩哔哩 `#fb7299`、小红书 `#fe2c55`
- Hover 时 `boxShadow` 发光

### 5.4 DetailView（`DetailView.tsx`）

- 全屏玻璃拟态弹窗
- 背景 `rgba(10,10,11,0.4)` + `backdropFilter: blur(48px)`
- 图片 `objectFit: contain` 完整显示
- 底部渐变遮罩 + 居中标题（Playfair serif italic） + 描述文字（散文诗排版，行高 2.6，zinc 灰）
- 底部进度条 + 缩略图导航
- 左右热区 + 方向键 + 滚轮切换
- ESC 关闭

### 5.5 MusicPlayer（`MusicPlayer.tsx`，右下角悬浮）

- 固定右下角圆形按钮
- 点击展开毛玻璃播放器面板
- 播放/暂停 + 进度条拖动 + 时间显示
- 循环播放 `情歌.mp3`
- 播放时按钮旋转动画
