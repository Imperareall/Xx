## 四、背景纹理与全局视觉体系

```css
body {
  background: #0c0c0e;
  color: #a1a1aa;
```

### 4.2 环境光晕（Ambient Glow）

通过 `body::before` 和 `body::after` 固定伪元素，飘浮在页面上方/下方：

```css
body::before {
  position: fixed; top: -20%; left: -10%;
  width: 700px; height: 700px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(99,102,241,0.06) 0%, transparent 70%);
  filter: blur(140px);
  pointer-events: none;
  z-index: -1;
}
body::after {
  position: fixed; bottom: -20%; right: -10%;
  width: 800px; height: 800px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(139,92,246,0.05) 0%, transparent 70%);
  filter: blur(160px);
  pointer-events: none;
  z-index: -1;
}
```

### 4.3 胶片噪点（Film Grain）

```css
.film-grain {
  position: fixed; inset: 0; z-index: 9998;
  pointer-events: none;
  opacity: 0.025;
  mix-blend-mode: overlay;
  background-image: url("data:image/svg+xml,...feTurbulence...");
  background-repeat: repeat;
  background-size: 256px 256px;
}
```

用了 SVG `feTurbulence` 噪声，`baseFrequency="0.82"` + `numOctaves="5"`，`opacity: 0.025` 几乎不可见，仅为银盐胶片质感。

### 4.4 多米诺骨牌画廊背景光斑

`GridView.tsx` 的 3D 画廊区域有两个缓慢漂浮的径向渐变光斑（indigo + purple），动画循环 18s/24s。

## 五、各页面功能详解

### 5.1 Memory 页面（`GridView.tsx`）

**顶部 Hero**：
- 全屏封面图（`/合照pic/微信图片_202606110934332.jpg`），`objectFit: contain`
- 毛玻璃居中卡片（subtitle + 标题 + Since）
- 底部 Scroll 指示器呼吸动画
- 背景视差上移效果（useScroll + useTransform）

**主标题**：
- "虚虚小心" 使用 `<ParticleText>` 微光粒子悬浮 + **白银金属渐变**（`linear-gradient(135deg, #b8c6db, #e8ecf1, ...)`）+ `diamond-shimmer` 4秒循环流光 + 多层发光 shadow + Playfair Display italic 300

**副标题**：
- "一整个宇宙 换一颗红豆" 同样使用 `<ParticleText>` 粒子悬浮 + 白银渐变 + 钻石流光 + 立体 shadow + Playfair Display italic 300（`clamp(24px, 4vw, 40px)`）

**3D 多米诺骨牌画廊**：
- 9 张骨牌卡片从左到右以 `position: absolute` 排列
- 每张卡片有独特 3D 倾斜预设（`tiltVariants` 数组，9 种不同 rotateX 值）
- 透视容器 `rotateX: 22deg, rotateY: -15deg, rotateZ: -5deg`
- 卡片间距：190px（左/上偏移渐变递减）
- **滚轮驱动**：画廊可横向滑动，原生 `el.addEventListener('wheel', ...)` 阻止页面上下滚
- **Hover 动效**：卡片上浮 y: -60 + scale 1.1 + zIndex 9999，其余卡片 brightness 0.3 模糊
- 卡片半透明边框 `border: 1px solid rgba(255,255,255,0.05)`
- 阴影 `0_30px_60px_-15px_rgba(0,0,0,0.9)`，hover 时加深
- **点击**：触发 `openDetail(item.id)`，弹出 DetailView

### 5.2 About 页面（`AboutSection.tsx`）

**纯图片滚动流，无文字内容**：

顺序排列 4 条横向自动滚动的图片条：

1. **Forever** — 向左滚动（73张，`autoDuration = Math.min(Math.max(images.length * 1.2, 20), 120)`）
2. **experiencepic** — 向左滚动（21张）
3. **xx** — 向右滚动（26张）
4. **Pets** — 向右滚动（13张）

每条图片流使用 `ImageStrip` / `PetsStrip` 组件实现：
- 左右渐变遮罩
- Hover 时 `whileHover: scale(1.05)`
- 点击使用 ImageLightbox 多图浏览
- 背景色 `#0F1115`
- **已去除 `drag="x"`**（避免与自动滚动冲突导致暂停）
- 图片列表通过 `src/hooks/useImages.ts` 的 `import.meta.glob` 自动扫描

About 页面图片点击后的灯箱：

- 底部圆点进度条
- ESC / ←→ / 滚轮切图

---

### 5.6 MusicPlayer（`MusicPlayer.tsx`）

- 固定右下角圆形按钮
- 点击展开毛玻璃播放器面板
- 播放/暂停 + 进度条拖动 + 时间显示
- 循环播放 `/情歌.mp3`
- 播放时按钮旋转动画

---

### 5.7 ParticleText（`ParticleText.tsx`）

通用 React 组件，将文字转化为 canvas 粒子动画：
- `font` prop — 字体样式（如 `300 italic clamp(...) 'Playfair Display', serif`）
- `gradient` prop — CSS gradient（传入则文字显示渐变，否则透明占位）
- `textFilter` prop — CSS filter（drop-shadow）
- `className` prop — 动画类名（如 `shimmer-text`）
- 采样：每像素，离屏 canvas → `getImageData`
- 动画：鼠标排斥 + 弹簧回弹 + 呼吸闪烁 + 微型光晕
- 粒子尺寸：0.5-1.2px
